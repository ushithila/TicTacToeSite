TicTacToe:The simplest of games

This project is a game that we have created in a program called pygame. It is a generic verison of Tic Tac Toe that is player vs. bot designed. The bot will always do the best possible move to win. It is basically playing Tic Tac Toe on hard mode. We created this because Tic Tac Toe is an all around fun game that is quick, simple, and easy. Anyone can play Tic Tac Toe and thats why we decided to make our own generic verison. The creators of this project are Jay Singh, Alisa Aleksandrova, and Shitila Urshi.
